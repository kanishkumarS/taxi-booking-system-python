# 🚖 Taxi Booking System (Python)

## 📌 Description
This project is a simulation of a taxi booking system built using Python. It allocates taxis to passengers based on distance, availability, and revenue optimization, similar to real-world ride-booking platforms.

## 🛠️ Tech Stack
- Python
- Data Structures (Lists, Dictionaries)

## ⚙️ Features
- Dynamic taxi allocation based on nearest distance
- Revenue tracking for each taxi
- Passenger rejection based on maximum pickup radius
- Time calculation for travel and waiting
- Priority handling based on availability and earnings

## 🧠 How It Works
- Each taxi has attributes like current position, revenue, and availability time
- Passenger requests are processed sequentially
- The system finds the nearest available taxi
- If multiple taxis are available, the one with lower revenue is selected
- Travel cost is calculated based on distance and pricing rules

## 📚 Key Concepts Used
- Dictionary-based data management
- List manipulation and indexing
- Time calculation logic
- Conditional decision-making
